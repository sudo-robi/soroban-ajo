# Soroban Ajo – Storybook

Welcome to the UI component library for **Soroban Ajo** (Drips).

## What’s here

- **Components** – Reusable UI components (GroupCard, ProfileCard, Icon, etc.) with stories and controls.
- **Design system** – Tailwind-based tokens and shared styles (see Canvas for full styling).

## Running Storybook

- **Dev:** `npm run storybook` (port 6006)
- **Build:** `npm run build-storybook` (static build in `storybook-static`)

## Adding stories

Add new stories under `stories/` or next to components as `*.stories.tsx`. Use the same `@/` path alias as the app.
